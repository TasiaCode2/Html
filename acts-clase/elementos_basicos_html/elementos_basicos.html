Elementos básicos de HTML

Además de los elementos vistos en clase, también existen otras etiquetas básicas.


<br> para saltos de línea.

Los encabezados <h1> ... <h6> para títulos dentro del contenido.

<ol> para listas ordenadas y <ul> para listas desordenadas.

<li> para elementos dentro de la lista.

<script> se utiliza para insertar o hacer referencia a un script ejecutable.

<link> especifica la relación entre el documento actual y un recurso externo. Es muy común utilizarlo para aplicar estilos al documento por medio del atributo rel="stylesheet".
