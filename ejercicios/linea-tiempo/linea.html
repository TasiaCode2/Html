Línea del tiempo sobre HTML y JS

Año Evento
1989
  • Tim Berners-Lee, un científico de la computación británico, inventa la World Wide Web y crea el primer lenguaje de marcado llamado HTML (HyperText Markup Language)
1991
  • Se publica la primera especificación de HTML, conocida como HTML 1.0. Incluía elementos básicos como encabezados, párrafos y listas
1993
  • Se lanza HTML 2.0, que incluía soporte para tablas y formularios
1995
  • Se lanza HTML 3.0, que agregó soporte para estilos y hojas de estilo en cascada (CSS)
  • Se lanza JavaScript, un lenguaje de programación de alto nivel diseñado para ser utilizado en el lado del cliente en las aplicaciones web. JavaScript se convierte rápidamente en una herramienta clave para la creación de sitios web dinámicos e interactivos.
1997
  • Se lanza HTML 4.0, que incluía soporte para marcos (frames), soporte para scripts de lado del cliente (JavaScript) y otros nuevos elementos y atributos.
1999
  • Se publica XHTML 1.0, una versión más estricta de HTML que se basa en XML y es compatible con JavaScript.
2000
  • Se publica HTML 4.01, una versión actualizada de HTML 4.0 que corrige algunos errores y hace algunos cambios menores
2004
  • Se publica XHTML 2.0, que se centra en la modularidad y la estructura de la semántica del documento. Sin embargo, no se adopta ampliamente debido a su falta de retrocompatibilidad.
2005
  • Ajax (Asynchronous JavaScript and XML) se vuelve popular como una forma de crear aplicaciones web dinámicas y mejorar la interactividad con el usuario. Permite a las páginas web actualizar ciertas secciones sin recargar toda la página.
2008
  • La W3C publica HTML5, que incluye nuevos elementos semánticos, mejoras en la accesibilidad y características de almacenamiento local. También se incluye soporte para contenido multimedia como video y audio.
2009
  • jQuery, una biblioteca de JavaScript que simplifica la manipulación del DOM y el manejo de eventos, se convierte en una herramienta popular entre los desarrolladores web.
2010
  • Lanzamiento de AngularJS, un framework de JavaScript de código abierto para el desarrollo web de aplicaciones de página única (SPA).
2014
  • La W3C (World Wide Web Consortium) publica la recomendación oficial de HTML5
2015
  • Se lanza ECMAScript 6 (también conocido como ES6 o ECMAScript 2015), una actualización importante del estándar de JavaScript que incluye nuevas características como las funciones de flecha, las clases y los módulos.
2016
  • La W3C anuncia que HTML5 es la "recomendación oficial" y que ya no se están desarrollando nuevas versiones de HTML
  • Lanzamiento de Angular, una reescritura completa de AngularJS, con una arquitectura más modular y basada en componentes.
  • Lanzamiento de Vue.js, un framework de JavaScript progresivo y reactivo para el desarrollo de interfaces de usuario.
2017
  • Se lanza React, una biblioteca de JavaScript de código abierto desarrollada por Facebook para la creación de interfaces de usuario. Se vuelve popular entre los desarrolladores debido a su eficiencia y facilidad de uso.
2019
  • La W3C publica la primera versión de HTML Living Standard, que es un documento en evolución continua que describe el estado actual de HTML y su desarrollo futuro.
