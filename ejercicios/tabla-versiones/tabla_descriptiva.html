Tabla de versiones

Versiones de HTML

Versión Características
HTML Versión original de HTML, utilizada para marcar documentos y páginas web. Contenía sólo unos pocos elementos, como HEAD, TITLE, y BODY.
HTML 2.0 Se introdujeron nuevos elementos como IMG, y se añadió soporte para formularios. También se mejoró la estructura y organización del documento.
HTML 3.2 Se añadieron nuevos elementos y atributos para mejorar la accesibilidad y la estructura semántica del documento, incluyendo elementos para listas y tablas.
HTML 4.0 Se añadieron soporte para hojas de estilo y scripts del lado del cliente (JavaScript), además de nuevos elementos y atributos. También se mejoró el soporte para contenido multimedia.
XHTML 1.0 Una versión más estricta de HTML que se basa en XML y es compatible con JavaScript. Se requiere que todos los elementos estén cerrados y todos los atributos se escriban en minúsculas.
HTML5 Se introdujeron nuevos elementos semánticos, como HEADER, FOOTER y ARTICLE, así como nuevos atributos para mejorar la accesibilidad. También se añadió soporte para contenido multimedia como video y audio, y para almacenamiento local.

Versiones de JS

Versión Características
JavaScript Versión original de JavaScript, utilizada para agregar interactividad y funcionalidad a las páginas web. Permitía la manipulación de elementos del DOM y la creación de scripts de lado del cliente.
ECMAScript 3 Estándar que estableció una serie de reglas para la implementación de JavaScript en navegadores web y otras aplicaciones. Incluía nuevas características como las expresiones regulares y las funciones anónimas.
ECMAScript 5 Incluía nuevas características como Object.create() y JSON.parse(), así como mejoras en la manipulación de cadenas y arreglos. También se introdujo el "modo estricto" para una codificación más segura y predecible.
ECMAScript 6 (ES6) Actualización importante del estándar de JavaScript que incluye nuevas características como las funciones de flecha, las clases y los módulos. También incluye mejoras en la manipulación de cadenas y arreglos, y soporte para promesas y generadores.
ECMAScript 2016 (ES7) Incluye nuevas características como Array.prototype.includes(), el operador de exponenciación y las propiedades de Array.prototype y Object.
ECMAScript 2017 (ES8) Incluye nuevas características como async/await para simplificar la programación asíncrona, y Object.entries() y Object.values() para trabajar con objetos de manera más sencilla.
